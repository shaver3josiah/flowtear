**PhaseBadge** — the prominent current-phase indicator; use on the Today header and atop the log sheet.

```jsx
<PhaseBadge phase="fertile" subtitle="Day 13" />
<PhaseBadge phase="menstrual" subtitle="Day 2 of ~5" />
```

Phase-tinted pill (soft background + matching text) with a color dot. Phases: `menstrual|follicular|fertile|ovulation|luteal`. For small inline status, use `Badge` with the same tones.
