**IconButton** — a round, icon-only tap target; use for toolbar actions, calendar arrows, and close/overflow buttons.

```jsx
<IconButton label="Previous month" variant="ghost"><ChevronLeft/></IconButton>
<IconButton label="Add" variant="primary"><Plus/></IconButton>
```

Variants `soft|ghost|primary`; sizes `sm(36)|md(44)|lg(52)`. Always provide `label`. Pair with a CDN icon set (the system uses Lucide — see README → Iconography).
