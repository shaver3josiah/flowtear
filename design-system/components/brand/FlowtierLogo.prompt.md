**FlowtierLogo** — the Flowtier brand mark, a smiling pink alarm clock (time + cycles, made friendly); use as the app icon, splash/onboarding hero, or beside the Wordmark.

```jsx
<FlowtierLogo size={120} />
```

Pure SVG, theme-tinted from `--primary` / `--primary-strong` / `--deep`, so it re-colors with the active preset. It sits beautifully on the pink-flow, flower-flecked field shown in the "Logo — the Flowtier clock" brand card. For the plain bloom motif (empty states, loaders), use `FlowerMark`.
